# Billy Labs Master Plan

## Current Goal
Build the private Curator Studio before adding major new public features.

## Phase 1 – Understand the Current Project
- Inspect the latest stable project ZIP.
- Document the current architecture.
- Identify where mashups, images, blurblets, favorites, and other data originate.
- Determine the best integration points for metadata and AI.

Deliverable: Architecture Report.

## Phase 2 – Curator Data Bible
Define the long-term metadata model, search behavior, AI workflow, verification, batch editing, and curator rules.

## Phase 3 – Database Design
Design metadata, AI suggestions, review history, locks, and audit trail.

## Phase 4 – Curator Studio
Design the Metadata Editor, AI Suggestions, Review Queue, Research Lab, Search, Batch Operations, Statistics, and Ideas Engine.

## Phase 5 – AI Integration
AI suggests metadata and inconsistencies but never overwrites curator-approved data automatically.

## Phase 6 – Ideas Engine
AI proposes achievements, exhibits, collections, hidden themes, and interesting discoveries.

## Workflow
Design → Review → Implement → Test → Release.

## Immediate Next Task
Inspect the current Billy Labs project and produce an Architecture Report.
