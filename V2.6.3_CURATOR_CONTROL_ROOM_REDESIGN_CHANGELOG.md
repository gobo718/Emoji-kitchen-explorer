# Billy Labs v2.6.3 — Curator Control Room Redesign

## Source of truth
Built from `Billy-Labs-v2.6.2-documentation-consolidation-review.zip`.

## Changes
- Replaced the existing tab-first Curator opening with a Control Room landing page.
- Added a CSS-rendered purple laboratory door; no external image dependency was introduced.
- Added enough explanatory copy for a substitute curator to orient themselves without turning the page into a manual.
- Established four departments: Collections & Publishing, Discovery Engine, Diagnostics, and Settings.
- Renamed the existing Research Lab interface to Discovery Engine.
- Kept Combo Lookup functional inside Discovery Engine.
- Reserved the Discovery Engine for future AI analysis, metadata review, rare-trait research, collection suggestions, preview, bulk apply, and undo.
- Kept all existing collection editing and publishing behavior unchanged.

## Safety boundary
No AI feature, metadata mutation, or bulk operation was implemented in this release. The interface only establishes the correct architectural home for them.
