# Billy Labs v2.4.3aa — Final Home Polish Handoff

Use **Billy-Labs-v2.4.2-navigation-home-layout.zip** as the only source of truth.

This is a narrowly scoped polish update. Do not redesign the Home page, navigation, discovery system, collection behavior, blurblet voting section, or any other page.

## 1. Tighten the Daily Pick card

The Daily Pick card currently leaves too much empty space beneath its blurblet.

Required behavior:

- Keep the Daily Pick blurblet exactly where it is.
- Keep the blurblet **right-aligned**.
- Keep the card height adaptive so longer blurblets can expand the card naturally.
- Remove the large fixed empty area beneath short blurblets.
- Measure the current blurblet font size or line height.
- Leave approximately **one blurblet line-height** of padding between the final line of the blurblet and the bottom border of the Daily Pick card.

Do not change the horizontal position of the blurblet.

## 2. Make “Write a Blurblet” usable directly on Home

The lined blurblet area should become an operable text-entry field.

Required behavior:

- The user can type directly into the blurblet area.
- Preserve the current visual style of the lined writing card.
- Keep the random mashup/Post-it reference visible.
- Remove the **Open in Studio** button entirely.
- Add a **Submit** button beneath the writing area.
- The Submit button must be **right-aligned**.
- Style the Submit button like the existing copy/action buttons:
  - compact rounded rectangle
  - colorful gradient
  - bold white text
  - visually consistent with the current Home page buttons

## 3. Add a submit confirmation popup

Pressing Submit should not publish immediately.

Show a confirmation popup asking whether the user is ready to submit or wants to continue working in Studio.

Suggested wording:

**Ready to submit your blurblet?**

Actions:

- **Submit Now**
- **Work on It More in Studio**
- **Cancel**

Behavior:

- **Submit Now** continues the submission flow.
- **Work on It More in Studio** opens Studio and carries over:
  - the selected mashup
  - the full text already entered
- **Cancel** closes the popup and preserves the draft on Home.

The user must not lose any typed text when opening Studio or cancelling.

## 4. Preserve all existing v2.4.2 layout decisions

Do not change:

- Six persistent bottom navigation icons:
  - Home

  - Studio
  - Gallery
  - Profile
  - Trophies
- Daily Pick blurblet right alignment
- Surprise Me and Newly Created side-by-side

- Vote on a Blurblet as a full-width section
- Write a Blurblet as a full-width section beneath it
- Daily Pick auto-collection
- Existing copy/favorite button behavior
- Existing mashup discovery and navigation rules
