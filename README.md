# Billy Labs v2.6.1

Curator Foundation hotfix built on v2.6.0.

## What changed
- Public GitHub Pages visitors no longer receive the Curator launcher automatically.
- Curator mode still works when deliberately enabled with `?curator=1`, and remains available on approved local/development hosts.
- The Curator control room now provides prominent, direct access to the Museum Archive.
- Visible release labels and Curator backup metadata now report v2.6.1.

## Curator Explorer / Museum Archive
The former public Explorer is an internal Museum Archive for curator inventory search, blurblet-status filtering, inspection, and editor handoff.
