# Curator Explorer Behavior Contract

1. Curator Explorer is not linked from visitor bottom navigation.
2. All-status view can browse the complete inventory.
3. Missing means neither draft nor published content contains a non-empty blurblet.
4. Draft means a private draft contains a non-empty blurblet.
5. Published means published curator content contains a non-empty blurblet.
6. Published takes display precedence when both published and draft content exist.
7. Recently Added means rows belonging to the newest release date in the current inventory snapshot.
8. Inspector editing opens the existing mashup curator panel; this foundation does not duplicate editor logic.
9. No bulk mutation is performed from Curator Explorer.
